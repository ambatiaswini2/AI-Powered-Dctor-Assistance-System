# AI-Powered Doctor Assistance System

## Description
AI-Powered Doctor Assistance System using Deep Learning and Computer Vision that analyzes medical images (e.g., X-rays) using a Convolutional Neural Network (CNN) to predict diseases with confidence scores. The system also uses Grad-CAM to highlight important regions in the image, improving interpretability. It includes a chatbot-based interface for real-time interaction and preliminary diagnosis support.

## Features
- Medical image analysis using CNN  
- Disease prediction with confidence score  
- Grad-CAM visualization (Explainable AI)  
- Chatbot interface for interaction  
- Report generation and analysis  

## Tech Stack
- Python  
- TensorFlow / Keras  
- OpenCV  
- HTML, CSS, JavaScript  

## How to Run

1. Clone the repository:
git clone https://github.com/ambatiaswini2/AI-Powered-Doctor-Assistance-System.git

2. Navigate to the project folder:
cd AI-Powered-Doctor-Assistance-System

3. Install dependencies:
pip install -r requirements.txt

4. Run the application:
python app.py

5. Open in browser:
http://localhost:5000

## Model Details
- Model: Convolutional Neural Network (CNN)  
- Evaluation Metrics: Accuracy, Precision, Recall, F1-score  
- Explainability: Grad-CAM
